Git & GitHub Cheat-Sheet (Beginner Guide)

Basic Commands

  1. Check if you are in githubdemo folder: cd githubdemo
  2. Go into any repository folder: cd folder_name
  3. Exit from repo: cd ..
  4. Create new repo folder: mkdir newrepo
  5. Check status: git status
  6. Add changes: git add .
  7. Commit changes: git commit -m "your message"
  8. Check status again (nothing to commit means clean)

Pushing Local Repo to GitHub

  1. Create a repository on GitHub first
  2. Add remote: git remote add origin <repo-link>
  3. Verify remote: git remote -v
  4. Check current branch: git branch
  5. Push to GitHub: git push -u origin main

Whenever you make changes:
git add .
git commit -m "message"
git push

Learning Branches & Merge

Work on a new branch, push it to GitHub, create a Pull Request, resolve conflicts if any, and merge into main.

Merging Using Pull Request

  1. Push branch: git push -u origin newbranch
  2. Open GitHub repository
  3. Click Compare & Pull Request
  4. Base: main, Compare: newbranch
  5. Resolve conflicts if shown
  6. Merge Pull Request
