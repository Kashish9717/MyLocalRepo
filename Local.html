Git Commands Reference

  1. Check if you are in githubdemo folder: cd githubdemo
  2. Go into any repository folder: cd folder_name
  3. Exit from repo: cd ..
  4. Create new repo folder: mkdir newrepo
  5. Check status: git status
  6. Add changes: git add .
  7. Commit changes: git commit -m "your message"
  8. Check status again: git status (nothing to commit means ready)

Pushing to GitHub

  1. Initialize: create a repo on GitHub first to get a link
  2. Set remote: git remote add origin <link>
  3. Verify remote: git remote -v
  4. Check branch: git branch
  5. Push to GitHub: git push -u origin main

If you create new files, just do git add, commit, and git push.
